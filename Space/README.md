# REST Interface

| HTTP verb | Data operation |
| --------------- | --------------- |
| GET | Retrieve |
| POST | Create |
| PUT | Update (update all fields and use default values for any undefined fields) |
| DELETE | Delete |
| MERGE | Update (update only the fields that are specified and changed from current version) |
